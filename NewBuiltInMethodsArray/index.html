Array

Novos métodos para maniputalção de Arrays.

Array.from

Transforma elementos que são similares ao array, em array, possíbilitando utilizar todos os métodos do prototipe Array.

Array.find e Array.findIndex

Find: Procura por um dado dentro de um array e retorna o primeiro que estiver de acordo com a condição informada.

FindIndex: Procura a posição de um dado dentro do array.

Array.fill

Com este métodos podemos preencher valores dentro de um array, seja total ou em determinadas posições.

Array.off

Permite criar um array através de certos elementos.
